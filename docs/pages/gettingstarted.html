Welcome to B10ggy! A simple SSG for creating simple blogs and websites like the one your are on now. Asumming you had already read the readme, lets create our first project! It's really easy to get up and running, just open up a command prompt and run path/to/bloggy.exe create [project name] and it should create a simple project for you!
