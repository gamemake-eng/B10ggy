Unlike other JAM stack frameworks, it is very simple to add your own styling and layout to your sites! Here are some of the basics.

Templates


Templates are basicly skeletons for your posts and homepage. They are just basic html files. That means that you can style them using css, which is pretty awesome. You can basicly just create an inline style tag and do all of your css magic in there, but if you want them in seprate files you have to do some extra work which we will not be talking about right now. So maybe take a look in the home.html or post.html files and tinker a bit. Make sure to build your site every time you change something! NOTE: When using %% in css, make sure to type %%%% and not just a single %% or else you will get an error.

Tags


No, not html tags, I am talking about the Bloggy tag system. You are able to put anything in your post settings and refernece it with a tag like @this. Here is an example below.
settings.json
{
		"posts":[
			{
				"body":"pages/howimadeamericafat.html",
				"title":"Getting Started",
				"face":"UwU"
			}
		],
		"homepage_template":"home.html",
		"post_template":"post.html",
		"author":"Jow Byeden",
		"blog":"Murica"
}
